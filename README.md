# 消費税と主要税収・社会保障費の推移分析

このリポジトリは、日本の**消費税・法人税・所得税の推移**と、**社会保障費との関係**を分析したJupyter Notebookです。

---

## 📊 分析内容

| 内容 | 説明 |
|------|------|
| 消費税の税収推移 | 2000年〜2023年の推移を可視化 |
| 社会保障費の推移 | 年金・医療・介護などの支出と比較 |
| 消費税の税収割合 | 総税収に対する割合を算出 |
| 法人税・所得税との比較 | 景気との関連を視覚的に検証 |
